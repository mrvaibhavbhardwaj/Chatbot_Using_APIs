# Chatbot_Using_APIs
This project is an AI-powered multi-functional platform built using NVIDIA APIs that integrates multiple intelligent tools into a single system. It provides capabilities such as document analysis, chatbot interaction, code generation, and multilingual translation (12+ languages).
✨ Features
📄 Document Analysis
Upload and analyze documents (PDF, text, etc.)
Extract key insights and summaries
Intelligent question-answering from documents
🤖 AI Chatbot
Conversational AI interface
Context-aware responses
Supports general queries and domain-specific interactions
💻 Code Generation
Generate code snippets in multiple programming languages
Assist in debugging and optimization
Developer-friendly interface
🌍 Multilingual Translation
Supports 12+ languages
Real-time translation
High accuracy using AI models
🛠️ Tech Stack
Frontend: (Figma UI / Web UI)
Backend: Python / Node.js
APIs: NVIDIA AI APIs
Models: LLM-based models (via NVIDIA endpoints)
🏗️ System Architecture
User Interface (Frontend)
        │
        ▼
 Backend Server (API Handling)
        │
        ▼
 NVIDIA AI APIs (LLMs / Models)
        │
        ▼
 Processed Response to User

⚙️ Installation & Setup
1️⃣ Clone the Repository

Add API Key
Create a .env file and add:
